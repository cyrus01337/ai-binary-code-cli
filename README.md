# AI Binary Code CLI

CLI tool to act as a codec for binary code to ASCII plaintext and vice versa
written completely with AI to test not only my understanding of Go, but to
explore the extents of AI generated code as of July 21st 2026.

### Usage
```sh
# Convert text to binary
go run . --encode Hello, world!

# Convert binary back to readable text
go run . --decode TODO

# Test application
go test
```
